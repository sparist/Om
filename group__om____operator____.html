Operator
Program

An atomic Program element defined by any UTF-8 string. More...

Interpretation
A Program is interpreted as an Operator by treating it as containing as a single atomic UTF-8 string element containing the encoded text of all elements.
Normalization
A Program is normalized as an Operator by encoding and merging all elements.
