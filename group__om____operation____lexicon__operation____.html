lexicon
Operation

Normalizes the Operand as a quoted Lexicon. More...

Implementation
Om::Operation::LexiconOperation

A Program is interpreted as a Lexicon as follows:

  • Each Operator is a key, indexing to the Operand to its right.
  • In the case of duplicate Operators, the prior is removed.
  • An Operator without an Operand indicates that the system mapping for the Operator is to be explicitly included.
  • An Operand without an Operator indicates the default mapping, which is used for any Operator not found in the Lexicon.
